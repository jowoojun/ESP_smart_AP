펌웨어

현재 펌웨어 버전

  v 3.02 (2017/07/12)

최신 펌웨어 버전

  v 4.02 (2017/07/28)

설명

  • IP주소 : 컴퓨터 네트워크에서 장치들이 서로를 인식하고 통신을 하기 위해서 사용하는 특수한 번호이다. 이 번호를 이용하여 발신자를 대신하여 메시지가 전송되고 수신자를 향하여 예정된 목적지로 전달된다.
  • 유동IP : 컴퓨터에 고정적으로 부여된 IP로 한번 부여되면 IP를 반납하기 전까지는 다른 장비에 부여할 수 없는 IP 주소.
  • 고정IP : 장비에 고정적으로 IP를 부여하지 않고 컴퓨터를 사용할 때 남아 있는 IP 중에서 돌아가면서 부여하는 IP.

설명

  • 텍스트 필드에 단어를 입력하여 리스트에 추가하면 해당 단어가 쓰인 웹 사이트 접속을 차단합니다.

설명

  • 6가지 카테고리에 포함된 웹 접속을 차단할 수 있습니다. 시간을 설정하여 인터넷 사용을 제한할 수 있습니다.

설명

  • 모두 허용 : 모든 기기가 공유기를 통해 인터넷을 사용할 수 있도록 합니다.
  • 등록된 주소만 허용(whitelist) : 리스트에 등록된 맥주소를 가진 기기만 인터넷을 사용할 수 있고, 등록되지 않은 기기는 공유기를 통한 인터넷 사용을 제한합니다.
  • 등록된 주소 차단(blacklist) : 리스트에 등록된 맥주소를 가진 기기의 인터넷 사용을 제한합니다.

설명

  • 포트포워딩 : 공유기에서 특정 통신포트를 개방하여 통신이 되도록 하는것 입니다. 등록시 설정한 IP주소와 포트로 외부통신이 가능하게 해줍니다.

설명

  • Twin IP : 공유기와 유선(무선)으로 연결된 기기중 MAC Address를 기준으로 공인 IP 주소를 직접적으로 부여하는 기능입니다.
  • DMZ : 내부 IP중 하나의 IP를 지정하여 공유기에 사용하는 공인IP의 모든 포트를 지정된 IP주소로 연결하는 기능입니다.

설명

  • 시스템로그 : 장비 및 네트워크 운영 과정에서 발생하는 모든 내용들이 발생시간과 함께 기록된 자료입니다.

설명

  • 펌웨어 버전 : 공유기의 시스템적인 버전이며 최신 버전으로 업데이트를 통해 개선된 서비스를 이용할 수 있습니다.
